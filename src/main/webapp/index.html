Aplicativo Exemplo de Uso de EJB

Observações:
- Tentar inserir com id=3 lançará uma exceção de "Menor de idade inválido".
- Inserir com id=4 fará com que o nome seja alterado, após a inserção, para "nome" + "alterado".
- Mensagens com palavrões não serão aceitas.
- Lembre-se que a cada reinício do servidor, os dados serão apagados.
